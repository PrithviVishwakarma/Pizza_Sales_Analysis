Project OverviewThis project involves a comprehensive data analysis of pizza sales to uncover actionable business insights. The analysis utilizes a dataset containing detailed sales records, which are processed to derive Key Performance Indicators (KPIs) and visualize sales trends.Project AssetsThe following files are included in the repository:pizza_sales.xlsx: The raw data file containing comprehensive transaction-level records for all pizza sales.  ALL_QUERIES.sql: A complete set of SQL queries used for data extraction, KPI calculation, and trend analysis.  PIZZA SALES SQL QUERIES Document.docx: A structured documentation file that provides the SQL queries alongside their corresponding output examples for clarity and verification.  Pizza Sales Analysis DB.pbix: The Power BI file containing the final dashboard, data modeling, and interactive visualizations built upon the analyzed dataset.  Analysis Components1. Key Performance Indicators (KPIs)The following metrics were calculated to assess business performance:Total Revenue: The overall monetary value generated from all pizza sales.  Total Orders: The total number of unique orders placed.  Average Order Value: The mean revenue generated per order.  Total Pizzas Sold: The cumulative quantity of pizzas sold.  Average Pizzas per Order: The average quantity of items included in a single order.  2. Analytical Trends & VisualizationsThe project explores several dimensions of the data:Sales Trends: Analyzed by month and by day of the week to identify peak business periods.  Category and Size Performance: Examined the percentage of revenue and total quantity sold across different pizza categories and sizes.  Top/Bottom Performing Pizzas: Identified the best and worst-performing pizzas based on total revenue, total orders, and total quantity sold.  How to Use This RepositoryData Processing: Use pizza_sales.xlsx as your primary data source.Database Queries: Refer to ALL_QUERIES.sql or the PIZZA SALES SQL QUERIES Document.docx to replicate the analysis or perform further data exploration within a SQL environment.  Visualization: Open Pizza Sales Analysis DB.pbix in Power BI Desktop to interact with the visual dashboard and explore the data dynamically.
